WHAT I STUDIED IN 2020

웹표준 사이트

웹 접근성 지침서2.0을 바탕으로 사이트를 제작하였습니다. 웹 표준을 준수한 사이트를 직접 제작하면서 장애인, 노약자를 포함한 모든 사용자가 동일한 조건에서 웹 사이트를 이용할 수 있도록 했습니다.

반응형 사이트

어떤 종류의 디스플레이로 접속해도 불편함이 없도록 미디어쿼리를 이용해서 반응형 웹 사이트를 제작했습니다. 시맨틱 태그를 사용하여 문서의 구조를 체계적으로 표현했습니다.

기업 사이트

기업에서 가장 많이 사용하는 스타일의 사이트를 제작하였습니다. 미디어쿼리를 활용해서 반응형 사이트로 제작했고, 시맨틱 태그를 사용하여 코딩했습니다. 구글의 웹 API를 사용하여 새로운 영화를 클릭했을 때 유튜브의 영상이 레이어 팝업으로 나타나도록 작업했습니다.

react

React를 이용하여 포트폴리오 사이트를 제작하였습니다. 컴포넌트를 활용하여 사이트를 완성하였고, SCSS를 사용하여 스타일을 지정했습니다.
