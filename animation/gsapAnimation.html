GSAP Animation</doc>

smartwebuiux.tistory.com

GSAP는 웹 브라우저에서 애니메이션을 구현하기 위한 자바스크립트 라이브러리입니다. 기존 CSS Animation이나 jQuery 애니메이션보다 탁월한 퍼포먼스를 발휘 할 수 있도록 최적화 된 애니메이션 전용 라이브러리입니다.

유형 메서드 설명
Method gsap.to() 일반적인 유형의 애니메이션을 정의합니다.
gsap.from() 일반적인 유형의 애니메이션을 반대로 정의합니다.
gsap.fromTo()
gsap.set()
gsap.timeline()
