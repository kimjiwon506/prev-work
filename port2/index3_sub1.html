React / Responsive / Webstandard를 활용한 포트폴리오 사이트입니다

Object01 Webstandard

웹 표준 지침서를 준수하고 크로스브라우징 작업을 한 사이트입 니다. 모든 사용자가 이용에 불편이 없도록 대체 텍스트로 웹 접 근성 또한 고려하였습니다. 슬라이드, 탭, 메뉴, 팝업, 동영상 기 능을 작업하였으며, 동적인 부분을 더하기 위해 slick.js 등과 같 은 플러그인을 이용하여 배너와 갤러리에 효과를 주었습니다. 크 로스 브라우징은 익스플로러 8버전부터 지원합니다.

Object02 Responsive

미디어 쿼리를 사용하여 여러가지 디바이스 종류에 따라 유연하게 보여지도록 반응형 작업을 진행한 사이트입니다. 웹표준과 동일한 슬라이드, 탭, 메뉴, 팝업, 동영상 기능등을 추가해서 작업을 진행하였고 오른쪽 컨텐츠에 3d Transition 효과를 주었습니다.
